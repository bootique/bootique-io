Features

10x faster than Springboot

Bootique has surpassed similar frameworks in numerous performance tests (see GitHub).

Explicit dependencies

With explicit modules and a modular structure, Bootique displays dependencies in plain view.

Bootique lets you create any kind of Java applications: web apps, RESTful, real-time streaming and ETL apps, and more.

CLI dispatch

Bootique offers a unique command dispatch feature that's rarely found in other frameworks.
